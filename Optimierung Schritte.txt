# Schritt 1 — Prüfen, dass nur ein Publisher aktiv ist

Bitte zuerst diese Befehle ausführen:

sudo docker exec -it mxck2_control bash -lc "source /opt/ros/humble/setup.bash && source /mxck2_ws/install/setup.bash && ros2 node list | sort"
sudo docker exec -it mxck2_control bash -lc "source /opt/ros/humble/setup.bash && source /mxck2_ws/install/setup.bash && ros2 topic info /autonomous/ftg/gap_angle -v"
sudo docker exec -it mxck2_control bash -lc "source /opt/ros/humble/setup.bash && source /mxck2_ws/install/setup.bash && ros2 topic info /autonomous/ftg/target_speed -v"
sudo docker exec -it mxck2_control bash -lc "source /opt/ros/humble/setup.bash && source /mxck2_ws/install/setup.bash && ros2 topic info /autonomous/ackermann_cmd -v"
Erwartung

Je FTG-Topic idealerweise:

1 Publisher für /autonomous/ftg/gap_angle
1 Publisher für /autonomous/ftg/target_speed
1 Publisher für /autonomous/ackermann_cmd aus deinem FTG-Control-Node

Wenn da mehrere stehen, müssen wir das zuerst bereinigen.
